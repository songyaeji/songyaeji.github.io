KITRI 화이트햇스쿨 1기 과정에서 수행한 인가된 교육용 공방 프로젝트. 공격을 직접
구현해봐야 효과적인 방어가 가능하다는 전제로, 게임 핵의 동작 원리를 공격자 관점에서
구현하고 방어자 관점에서 차단하며 게임 보안 전반을 다뤘다.

공격 측은 메모리 디버깅으로 핵심 주소를 찾아 메모리 값 변경·연산 조작·포인터 조작
방식의 핵 6종을 만들고 DLL Injection으로 묶었다. 방어 측은 API 첫 5바이트와 IAT
배열을 원본과 비교해 후킹을 탐지하고, 주입된 DLL과 디버거 attach를 실시간 감지해
게임 프로세스를 종료시키는 안티치트를 개발했다. 본인은 리버싱을 통한 게임 구조
분석에 참여하고 안티치트의 탐지 구조를 제안했다.

교육 과정 내 인가된 실습으로 진행됐으며, 코드는 공개하지 않는다.
